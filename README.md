# 🎙️ News Voice AI PRO

Sistema inteligente que transforma notícias do mercado financeiro em áudio com análise por IA.

## 🚀 Funcionalidades

- Scraping de notícias do Investing
- Análise com IA (resumo + sentimento)
- Conversão texto → áudio
- Dashboard web interativo

## 🧠 Tecnologias

- Python
- Flask
- OpenAI API
- gTTS
- BeautifulSoup
